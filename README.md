# ReVoltz Frontend

Frontend app for the ReVoltz battery second-life platform. Built with React + Vite.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Create a `.env` file in `frontend/` if needed:

```env
VITE_API_BASE_URL=http://localhost:8000
VITE_MODEL_API_BASE_URL=http://localhost:8001
```

3. Start the development server:

```bash
npm run dev
```

4. Open:

```text
http://localhost:5173
```

## Useful commands

```bash
npm run dev
npm run build
npm run lint
npm run preview
```
